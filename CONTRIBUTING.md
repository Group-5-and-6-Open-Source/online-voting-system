# Contribution Guidelines
[cite_start]To contribute to this project, please follow the Forking Workflow: [cite: 8, 14]

1. [cite_start]**Fork** the repository to your own GitHub account. [cite: 82]
2. [cite_start]**Clone** your fork locally. [cite: 77]
3. [cite_start]**Create a branch** for your feature: `git checkout -b feature/your-feature-name`. [cite: 80]
4. [cite_start]**Commit** your changes with clear messages. [cite: 116]
5. [cite_start]**Push** to your fork and submit a **Pull Request**. [cite: 83, 85]
6. [cite_start]A group **Code Reviewer** must approve the PR before the tutor is notified for merging. [cite: 86, 122]
