# Voter Registration (Local)

Quick start for the registration page added to this repository.

Prerequisites
- Node.js installed (14+)

Install and run

```powershell
cd c:\Users\ALABA\online-voting-system
npm install
npm start
```

Open the registration page in a browser:

- http://localhost:3000/register.html

Notes
- Submitted registrations are appended to `voters.json` in the project root.
- Uploaded photos are stored under `public/uploads` (multer auto-generated filenames).
